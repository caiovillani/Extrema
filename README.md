# Extrema

Projetos correlacionados ao desenvolvimento das ações profissionais do Dr. Caio V. em Extrema, MG.

## Objetivo
Este repositório atua como ponto central para organizar materiais, rascunhos e referências ligados às iniciativas do Dr. Caio V. em Extrema, Minas Gerais. A ideia é manter documentação breve sobre cada frente de trabalho conforme novas pastas e arquivos forem adicionados.

## Estrutura
- `README.md`: Visão geral do repositório e das iniciativas planejadas.
- (Futuro) Pastas específicas para cada projeto ou área de atuação.

## Contribuindo
1. Crie uma nova branch descrevendo a mudança.
2. Registre commits pequenos e objetivos.
3. Abra um pull request descrevendo o contexto e os resultados esperados.
