# Extrema

Projetos correlacionados ao desenvolvimento das ações profissionais do Dr. Caio V. em Extrema, MG.

## Sobre o repositório
Este repositório funciona como um ponto central para organizar estudos, rascunhos e documentos relacionados aos trabalhos desenvolvidos em Extrema. A ideia é manter as informações estruturadas e de fácil consulta para futuras referências.

## Estrutura sugerida
- **documentos/**: relatórios, anotações e referências de projetos.
- **dados/**: arquivos de dados brutos ou processados utilizados em análises.
- **scripts/**: códigos auxiliares para limpeza, análise ou geração de relatórios.

Sinta-se à vontade para adaptar a estrutura conforme as necessidades do projeto evoluam.

## Como contribuir
1. Crie uma nova branch para a alteração desejada.
2. Faça commits pequenos e descritivos.
3. Abra um pull request descrevendo claramente a motivação e o impacto das mudanças.
